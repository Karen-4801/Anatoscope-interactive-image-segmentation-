                                                   ANATOSCOPE 
                     Segmentación Interactiva de imágenes de angiografía

:::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::

Para ejecutar el código en su Matlab Online tenga en cuenta:

- Antes de ejecutar la aplicación, debe crear una carpeta en su MATLAB drive 
llamada "AA PROCESAMIENTO",y subir los archivos adjuntos en la carpeta del 
mismo nombre.

- Está diseñado para utilizarse en maltab online, de lo contrario, deberá editar
los paths utilizados en el código de "segmentation.mlapp".

::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::
 
Para usar la aplicacion correctamente:

1. Sube una imagen o elige una de nuestras imágenes de muestra.
2. Elija un método de umbralización (Utilice el control deslizante para ajustar el valor
de referencia si es necesario).
3. Elija un método de filtrado de contraste.
4. Elija un método de filtrado espacial (Utilice el control deslizante para ajustar el 
valor de referencia si es necesario).
5. ¡Segmentar!.
